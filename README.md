# Task Management App

## Features

### Authentication
- User registration and login.
- Only authenticated users can manage tasks.

### Task Management
- Fields:
  - `title` (required)  
  - `description` (optional)  
  - `status` (Pending / In Progress / Completed)  
  - `due_date` (optional)
- Users can:
  - Create tasks.
  - paginated list of own tasks.
  - Edit and update tasks.
  - Delete tasks.
  - Filter tasks by status, title, and due date range.

### UI / UX
- Messages for success/error operations.

### Bonus / Optional Enhancements
- AJAX-based create/update/delete without page reload.
